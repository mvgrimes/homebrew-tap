# Mvgrimes Tap

## How do I install these formulae?

`brew install mvgrimes/tap/<formula>`

Or `brew tap mvgrimes/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "mvgrimes/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
